Creating Tables

                <table>
                    <tr>
                        <th></th>
                        <th scope="col">Column 1</th>
                        <th scope="col">Column 2</th>
                    </tr>
                    <tr>
                        <th scope="row">Row Header</th>
                        <td>Table Data</td>
                        <td>Table Data</td>
                    </tr>
                </table>

Table Head, Table Body & Table Footer

                <table>
                    <!-- Table Head -->
                    <thead>
                        <tr>
                            <th></th>
                            <th scope="col">Column 1</th>
                            <th scope="col">Column 2</th>
                        </tr>
                    </thead>

                    <!-- Table Body -->
                    <tbody>
                        <tr>
                            <th scope="row">Row Header</th>
                            <td>Table Data</td>
                            <td>Table Data</td>
                        </tr>
                    </tbody>

                    <!-- Table Footer -->
                    <tfoot>
                        <tr>
                            <th>Table Footer</th>
                            <td> </td>
                            <td> </td>
                        </tr>
                    </tfoot>
                </table>
  • Section off long tables.
  • thead only contain table headings.
  • tbody contains all table data.
  • tfoot usually contain sums, differences, and other data results.

Column Span & Row Span

                <!-- Column Span -->
                <td colspan="2">This column spans 2 columns</td>

                <!-- Row Span -->
                <td rowspan="2">This row spans 2 rows</td>
  • Takes an integer (>= 1) to denote the number of columns/rows it spans across.

Header

                <!-- Method 1: header tag -->
                <header>
                    <!-- Navigation or introductory content -->
                </header>

                <!-- Method 2: div tag with header id -->
                <div id="header>
                    <!-- Navigation or introductory content -->
                </div>
  • Usually contains navigational links or introductory content.

Footer

                <footer>
                    <!-- footer content -->
                </footer>
    Usually contains:
  • Contact information
  • Copyright information
  • Terms of use
  • Site Map
  • Reference to page links

Section & Article

                <section>
                    <article>
                        <!-- body content -->
                    </article>
                </section>
    Section:
  • Used to split chapters, headings, topics, theme, etc.
    Article:
  • Contains the body of each section.

Aside

                <aside>
                    <!-- aside content -->
                </aside>
  • Used to mark additional information that isn't required in order to understand the main content.

    • Example:
  • Bibliographies
  • Endnotes
  • Comments
  • Pull quotes
  • Editorial sidebars

Attaching Images - Figure & Figcaption

                <figure>
                    <img src="image-link">
                    <figcaption>Image Description</figcaption>
                </figure>
  • figcaption - Adds an image description to the bottom of a figure.

Attaching Audio

                <audio controls>
                    <source src="audio-file-link/url" type="audio/mp3">
                </audio>
  • controls: shows the audio controls on the browser such as play and mute.

Attaching Video & Gifs

                <video src="video-file-link/url" controls>
                    Video not supported
                </video>

                <!-- Attaching Gifs -->
                <embed src="gif-file-link/url"/>
  • controls: shows the play/pause button, volume control and a fullscreen option on the browser.
  • autoplay: automatically plays the video as soon as page is loaded.
  • loop: loops/plays the video repeatedly.

Box-Sizing [Doc]

box-sizing: border-box;
  • border-box; - changes box model to include the padding, margin and borders within the element width specified.
  • By default, the padding, margin and borders add on to the element's width.

Border [Doc]

border: 1px solid black;
border: border-width border-style border-color;
  • border-width - e.g. 1px
  • border-style - e.g. solid
  • border-color - e.g. black

Border Radius [Doc]

Overflow [Doc]

overflow: scroll;
  • Controls what happens to content that overflows outside of its box.

Accessing different Selectors in CSS

Universal Selector [Doc]

                    margin: 0;

Visibility [Doc]

visibility: hidden;
    What's the difference between display: none and visibility: hidden?
  • display: none; - element is completely removed from web page.
  • visibility: hidden; - element is not visible on web page, but there's space reserved for it.

Positioning

Position [Doc]

position: static;
  • Changes the position of an element.
  • Values: static, relative, absolute, fixed, sticky

Z-Index [Doc]

z-index: 1;
  • Needs to be paired with any position values, besides static.
  • Element with larger Z-index is at the front.

Display [Doc]

display: inline;
  • Makes it possible for elements to be inline.

Float [Doc] & Clear [Doc]

float: right;
  • Floats elements as far left/right as possible.
clear: left;
  • Floats may allow multiple elements to float.
  • Clear defines how these float elements should behave when they bump onto each other.
